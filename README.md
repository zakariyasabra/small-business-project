# منصة دعم المشاريع الصغيرة

## Problem
توقف المشاريع الصغيرة وعدم وجود منصة لدعمها.

## Solution
إنشاء موقع يعرض المشاريع وطرق دعمها.

## Target Users
رواد الأعمال والمستثمرين.

## Features
- عرض مشاريع
- بحث
- إحصائيات
- فيديو
- نموذج تواصل

## Technologies
HTML5, CSS3, Bootstrap, JavaScript, jQuery

## How to Run
Open index.html in browser.
